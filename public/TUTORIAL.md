## 1. Главная страница сайта

Главная страница содержит:

- **Шапка** — заголовок статьи с количеством отзывов и датой обновления
- **Навигация** — быстрые ссылки на разделы сайта
- **Блок автора** — информация об авторе статьи с фото
- **Вступительный текст** — краткое описание темы
- **Критерии выбора** — на что обращать внимание при выборе курса
- **Список курсов** — карточки с кратким описанием курсов
- **Детали курсов** — подробная информация о каждом курсе
- **Контентные блоки** — дополнительная информация
- **FAQ** — часто задаваемые вопросы

## 2. Навигация по сайту

### Меню навигации

- Находится под шапкой
- Содержит ссылки на основные разделы
- Адаптивно работает на мобильных устройствах

### Якорные ссылки

- `#courses` — переход к списку курсов
- `#faq` — переход к FAQ
- Можно добавлять собственные якори в админке

## 3. Просмотр курсов

### Карточки курсов

Каждая карточка содержит:
- 🏷️ Бейдж (ТОП, Популярный, Новый)
- 🏫 Логотип школы
- 📝 Название курса
- 💰 Цена и старая цена (если есть скидка)
- 📅 Длительность
- 📄 Документ по окончании
- ✨ Ключевые особенности

### Подробности курса

При клике на курс открывается:
- Полное описание
- Список навыков
- Программа обучения
- Преподаватели
- Ссылки на отзывы
- Промокод (если есть)

## 5. Админ-панель

### Вход в админку

1. Перейдите по секретному URL: `/panel-x7k9m2`
2. Введите пароль администратора
3. После 5 неудачных попыток — блокировка на 15 минут

### Интерфейс админки

Админка разделена на вкладки:

| Вкладка | Описание |
|---------|----------|
| 📝 Контент | Редактирование мета-данных, заголовков |
| 👤 Автор | Информация об авторе |
| 📚 Курсы | Управление списком курсов |
| ❓ FAQ | Редактирование вопросов |
| 🧩 Блоки | Контентные блоки статьи |
| 📄 Страницы | Создание дополнительных страниц |
| ⚙️ Настройки | Favicon, навигация |

## 6. Редактирование контента

### Вкладка "Контент"

**Мета-данные** (SEO):
- `Title` — заголовок страницы (до 60 символов)
- `Description` — описание (до 160 символов)
- `Keywords` — ключевые слова через запятую
- `Canonical URL` — каноническая ссылка

**Статистика в шапке**:
- Количество отзывов
- Текст бейджа
- Подзаголовок

**Вступительный текст**:
- Редактор с поддержкой форматирования

### Вкладка "Автор"

- **Имя** — ФИО автора
- **Фото** — URL изображения или загрузка
- **Описание** — краткая биография
- **Ссылка** — URL профиля

## 7. Управление курсами

### Добавление курса

1. Перейдите на вкладку "Курсы"
2. Нажмите кнопку "Добавить курс"
3. Заполните все поля

### Поля курса

| Поле | Описание |
|------|----------|
| Название | Название курса |
| Школа | Название онлайн-школы |
| Логотип школы | URL логотипа |
| Ссылка | URL курса (партнёрская ссылка) |
| Цена | Текущая цена |
| Старая цена | Цена до скидки |
| Рассрочка | Ежемесячный платёж |
| Формат | Онлайн/Офлайн/Смешанный |
| Длительность | Срок обучения |
| Документ | Сертификат/Диплом |
| Для кого | Целевая аудитория |
| Бейдж | top/popular/new |

### Списковые поля

- **Особенности** — краткие преимущества (для карточки)
- **Навыки** — что получит студент
- **Преимущества** — подробные плюсы
- **Программа** — модули/темы курса

### Промокод

- **Код** — текст промокода
- **Описание скидки** — "Скидка 10%"
- **Процент** — числовое значение

### Ссылки на отзывы

Добавьте ссылки на отзывы с разных платформ:
- Платформа (Отзовик, IRecommend и т.д.)
- Количество отзывов
- Рейтинг
- URL

### Сортировка курсов

- Перетаскивайте курсы мышкой для изменения порядка
- Используйте кнопки ↑↓ на мобильных

### Массовая замена ссылок

1. Нажмите "Массовая замена URL"
2. Вставьте ссылки (по одной на строку)
3. Порядок соответствует порядку курсов

### Массовая замена промокодов

Формат: `ПРОМОКОД` или `ПРОМОКОД|Текст скидки|Процент`

```
SALE2024
PROMO|Скидка 15%|15
```

## 8. Управление страницами

### Создание страницы

1. Вкладка "Страницы" → "Добавить страницу"
2. Заполните slug (URL), заголовок, описание
3. Включите/выключите нужные блоки

### Настройки страницы

- **Slug** — часть URL (например: `java-courses`)
- **Заголовок** — для меню и мета-тегов
- **Показывать в меню** — вкл/выкл
- **Блоки** — какие секции отображать

### Контент страницы

Каждая страница может иметь:
- Свой заголовок
- Своего автора
- Свои курсы
- Свой FAQ
- Свои контентные блоки

### Дублирование страницы

Нажмите кнопку "Дублировать" для создания копии со всем контентом.

## 9. Экспорт и импорт данных

### Экспорт в JSON

1. Нажмите "Экспорт JSON"
2. Выберите страницу (главная или дополнительная)
3. Скачается файл `название-content-дата.json`

### Импорт из JSON

1. Нажмите "Импорт JSON" или перетащите файл
2. Выберите целевую страницу
3. Загрузите файл

### Формат JSON

```json
{
  "pageTitle": "Заголовок страницы",
  "metaData": {
    "title": "Title для SEO",
    "description": "Description",
    "keywords": "ключевые, слова",
    "canonicalUrl": "https://example.com/"
  },
  "author": {
    "name": "Имя Автора",
    "photo": "URL фото",
    "description": "Описание",
    "link": "URL"
  },
  "courses": [...],
  "faqData": [...],
  "contentBlocks": [...]
}
```

### Структура курса

```json
{
  "id": 1,
  "title": "Название курса",
  "school": "Школа",
  "schoolLogo": "URL логотипа",
  "url": "https://partner-link.com/",
  "price": 50000,
  "oldPrice": 100000,
  "duration": "6 месяцев",
  "format": "Онлайн",
  "document": "Диплом",
  "forWhom": "Новичкам",
  "features": ["Особенность 1", "Особенность 2"],
  "skills": ["Навык 1", "Навык 2"],
  "advantages": ["Преимущество 1"],
  "reviews": "Отличный курс!",
  "badge": "top",
  "promoCode": {
    "code": "SALE2024",
    "discountText": "Скидка 10%",
    "discountPercent": 10
  }
}
```

## 10. Публикация на хостинг

### Vercel (рекомендуется)

1. Подключите GitHub в настройках Lovable
2. Создайте репозиторий
3. Зайдите на [vercel.com](https://vercel.com)
4. Импортируйте репозиторий
5. Деплой автоматический

### Beget.ru

1. Экспортируйте JSON из админки
2. Склонируйте репозиторий: `git clone ...`
3. Соберите проект:
   ```bash
   npm install
   npm run build
   ```
4. Загрузите содержимое `dist/` на хостинг
5. Создайте `.htaccess`:
   ```apache
   RewriteEngine On
   RewriteBase /
   RewriteRule ^index\.html$ - [L]
   RewriteCond %{REQUEST_FILENAME} !-f
   RewriteCond %{REQUEST_FILENAME} !-d
   RewriteRule . /index.html [L]
   ```

## FAQ по админке

### Как сбросить пароль?
Пароль хранится в коде. Обратитесь к разработчику.

### Почему изменения не видны на сайте?
1. Нажмите "Сохранить" в админке
2. Обновите страницу сайта (Ctrl+F5)

### Как восстановить данные?
Используйте кнопку "Сброс к исходному" или импортируйте резервный JSON.

### Курсы не отображаются?
Проверьте, что у курсов заполнены обязательные поля (название, школа, URL).

### Изображения не загружаются?
1. Проверьте URL изображения
2. Используйте HTTPS ссылки
3. Убедитесь, что хост разрешает hotlinking
